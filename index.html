Events

We run the events that bring Chicago's coding community together. Multiple times a year we run daylong events for Chicago Public Schools students to teach students in a fun and engaging way. This past year we successfully ran event0 and will be running the Jones Jam in December.

Clubs

Chicode helps provide motivated computer science students and teachers with the resources to start their own clubs. With tools like Coding Workshops we give Chicago Public Schools' students an engaging and comprehensive curriculum to learn and teach code.
